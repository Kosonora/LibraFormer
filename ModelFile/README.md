# Folder Description 文件夹说明 
This folder contains model files generated by the Libraformer project. It includes the following files:
该文件夹为Libraformer项目产生的相关模型文件。包含以下文件：
* Documents Related to Machine Learning Models(机器学习模型相关文件)
* Documents Related to Deep Learning Models(深度学习模型相关文件)
## Machine Learning Model Files 机器学习模型文件
Machine learning model files are stored in the Joblib format and include both unsampled and resampled model files. 
The unsampled model file is named “No Resample Model.zip,” and the resampled model file is named “Resample Model.zip.”
机器学习模型文件采用Joblib格式储存，包含未重采样模型文件、重采样模型文件。未重采样模型文件为：No Resample Model.zip文件，重采样模型文件为：Resample Model.zip文件。
## Deep Learning Model Files 深度学习模型文件
The deep learning model files include the build files for the LibraFormer model, as well as files for other ablation experiments. Among these, LibraFormer.h5 is the LibraFormer model file, AutoEncoder.h5 is the model file for the autoencoder component of the LibraFormer model, and AutoDNN
is the deep learning model file for the ablation experiments. All deep learning models were built using TensorFlow.
深度学习模型文件包含LibraFormer模型的构建文件，以及其他消融实验文件。其中LibraFormer.h5为LibraFormer模型文件，AutoEncoder.h5为LibraFormer模型自编码器部分模型文件，AutoDNN
为消融实验的深度学习模型文件。深度学习模型均采用tensorflow构建。
