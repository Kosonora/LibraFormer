# Project Description 项目说明
## Basic Description 基本描述
In the context of real-world datasets, class imbalance is a common problem, which leads to higher misclassification costs for samples from minority classes in traditional classification methods.This project presents a framework—LibraFormer—that develops a strategy for addressing data classification tasks with class distribution imbalances.This approach achieves optimal data distribution through sample separation and majority class pruning;Build an encoder-decoder architecture to learn deep feature representations of the data;Build the LibraFormer model using a reused encoder architecture.This model integrates a generalized residual multi-head self-attention mechanism with a serial activation feedforward network, and enhances its nonlinear modeling capabilities through cross-layer feature fusion.

在真实场景数据集的背景下，类不平衡是一个普遍存在的问题，这导致在传统分类方法中，少数类样本的误分类成本升高。这个项目构建了一种策略用于应对类别分布不均衡的数据分类任务框架：LibraFormer。该方案通过样本分离和多数类裁剪实现最优数据分布；构建编码-解码架构以学习数据深度特征表示；复用编码器结构构建LibraFormer模型。该模型集成了泛化残差多头自注意力机制与串行激活前馈网络，并通过跨层特征融合提升了非线性建模能力。
